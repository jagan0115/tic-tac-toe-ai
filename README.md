# 🤖 AI Tic Tac Toe Game

An intelligent Tic Tac Toe game developed using Artificial Intelligence concepts.  
This project uses the **Minimax Algorithm** to make optimal decisions, ensuring the AI never loses.

---

## 🌐 Live Demo
👉 https://jagan0115.github.io/tic-tac-toe-ai/

---

## 📌 Features

- 🎮 Player vs AI mode  
- 👥 2 Player mode  
- 🧠 AI with Minimax Algorithm  
- 🎯 Difficulty levels (Easy, Medium, Hard)  
- 🔄 First move selection (Player / AI)  
- 🏆 Winner popup display  
- ✨ Modern UI with animations  

---

## 🧠 How It Works

The AI uses the **Minimax algorithm**, which:
- Explores all possible game states  
- Evaluates each move  
- Chooses the best optimal move  

👉 This ensures the AI plays perfectly and cannot be easily defeated.
